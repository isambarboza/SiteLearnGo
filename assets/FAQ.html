Perguntas Frequentes (FAQ)

O que é o LearnGo?
O LearnGo é uma plataforma projetada para conectar universitários através de caronas colaborativas, promovendo economia, sustentabilidade e interação social.
Como posso me cadastrar?
Para se cadastrar no LearnGo, você deve preencher os seguintes campos:
  • Nome Completo: Campo de texto para inserir seu nome completo.
  • Data de Nascimento: Campo de seleção para escolher sua data de nascimento.
  • RM: Campo de texto para inserir seu número de registro de matrícula.
  • Faculdade: Campo de seleção (dropdown) para escolher a faculdade onde estuda.
  • Curso: Campo de seleção (dropdown) para escolher o curso que você está realizando.
  • Senha: Campo de senha com requisitos mínimos de segurança.
  • E-mail: Campo de texto para inserir seu e-mail.
  • Endereço: Campo de texto para inserir seu endereço completo.
Após preencher todos os campos, você deve clicar no botão de cadastro para enviar o formulário.
O uso do LearnGo é gratuito?
Sim, o uso básico do LearnGo é gratuito. No entanto, podem existir opções premium para funcionalidades adicionais no futuro.
Como posso entrar em contato com o suporte?
Você pode entrar em contato com nossa equipe de suporte através do formulário de contato disponível na página "Contato" do nosso site.
